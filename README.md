# Features
Current list of features include: 
* Playing with any amount of other players, but without AI yet
* CLI functionality
* Automated tests and builds

Future features can be tracked via issues

# Installation

Requirements:

* Google test

Steps:
1. in the root of project ```cmake .```
2. ```make```
3. ```./src/blackjack_run```
4. Enjoy!
